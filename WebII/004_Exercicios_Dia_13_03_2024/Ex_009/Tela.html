9) Faça uma página Web que contenha um formulário HTML com os campos: Matrícula, Nome, Telefone, Endereço, Idade, e Curso. Os dados digitados no formulário deverão ser enviados para um programa escrito em PHP, que deverá chamar uma função para verificar se o aluno é maior ou menor de idade. Se o aluno for menor de idade (menor de 18 anos), o programa deverá escrever a seguinte mensagem na tela: “Aluno Menor de Idade – É necessário a assinatura do Pai ou Responsável!”, em cor vermelha . Se o aluno for de maior de idade, mostrar a mensagem: “Matrícula realizada com sucesso”, em cor azul.
